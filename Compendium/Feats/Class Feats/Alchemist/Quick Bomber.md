# Quick Bomber [](#Actions "Single Action") &emsp;*(Feat 1)*

[Alchemist](Alchemist.md "Class Trait")
**Description:** You keep your bombs and bomb-related reagents in easy-to reach pouches from which you draw without thinking.

**Activity:** Single Action  â¬¥

## Mechanical Rules

-  You get to do both of the following:
	- Interact to draw a bomb, versatile vial, or Use Quick Alchemy to create a bomb.
	- Strike with the bomb or vial.
- If you have the ability to create more than one bomb at a time with Quick Alchemy (such as from the double brew class feature), you can Strike with only one of the bombs you create with this action.
